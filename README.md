# howbig

```
$ howbig -h
Count the number of opaque pixels in an SVG

Usage: howbig [OPTIONS] <PATH>

Arguments:
  <PATH>  Path to SVG file

Options:
  -p, --percentage  Calculate opaque as a percentage
  -s, --square      Calculate opaque as a percentage of bounding square
  -h, --help        Print help
```
